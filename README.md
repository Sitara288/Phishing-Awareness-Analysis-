Phishing Awareness

This project explains phishing attacks and how to stay safe online. Phishing is a cyber attack where attackers try to steal sensitive information like passwords, OTPs, or bank details by pretending to be a trusted source.
 Key Points
Phishing uses fake emails, messages, or websites
Common types: Email phishing, SMS phishing, and website spoofing
Always check sender details and URLs carefully
Prevention
Do not click unknown links
Use strong passwords and enable 2FA
Verify before sharing personal information
Goal
To increase awareness about phishing attacks and promote safe internet usage.
